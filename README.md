### Objetivo
Criar um framework gráfico usnado como base xlib onde ira gerenciar lousas digitas.

### Requisitos
|Funcionais|Não Funcionais|
|----------|--------------|
|Conseguir escrever na lousa| Usar docker para disponibilizar o projeto |
|Ter mecanismos para apagar o que for escrito| 
|Mudar a cor do pinsel|
|Mudar entre claro e escuro o fundo da lousa|
|Sistema de cordenadas que mostre onde esta o pinsel|
|Consegur criar mais de uma lousa|
